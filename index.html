What does it actually do?

Using jQuery, Climb.js adds an animated, fixed position button to the document, fading in and out at certain scroll heights, that will return a user to the top of the current page when clicked.

How do I make it work?

It's pretty easy to use. Just add a button element with #climb to your existing HTML and include the JavaScript file after jQuery at the bottom of your document.

To call the Climb.js function use climb_init(); and then add your custom styles.

    
      <button id="climb"></button>

      <script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.2/jquery.min.js">
      <script src="scripts/climb.min.js">

      <script>
        climb_init('white', '#3F51B5', 'right', 0, 100, 800, 500);
      </script>

    </body>
    </html>

You can choose the color of the icon, the background color, which side of the window to display the button, x-coordinate, y-coordinate, the window height at which to display the button, and how long it takes to return to the top of the page.

Why?

The button Climb.js creates helps users to quickly return to the top of the current page. This is especially useful for really long, one-page websites.
